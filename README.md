#idea-マウスによるPC作業の快適さを追求する



・便座シートの貼って剥がせて洗えるものをマウスに貼る<br>
　→汗などをかいてしまっても、洗えるから清潔<br>
・好きなサイズに切れる<br>
　→どんなマウスにも対応<br>
・便座シートの下にクッションをいれる<br>
　→より手にフィット<br>
・腕と机の間にクッション
　→腕の疲れを防ぐ<br>
